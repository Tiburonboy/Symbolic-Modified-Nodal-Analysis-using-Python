25  Elliptic Function VCVS Filter

The circuit shown below is an elliptic function VCVS filter, which contains zeros and poles. The filter cutoff is 100 Hz. The circuit is from: Example 3-13 from Electronic Filter Design Handbook 3rd Edition.

This is the modified net list, V4 line was changed. The lines for the inductors were also changed.

C2 5 3 0.05e-6
C3 7 4 0.1e-6
